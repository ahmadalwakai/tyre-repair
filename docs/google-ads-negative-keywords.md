# Google Ads — Campaign-level negative keywords

Developer-facing reference for the TyreRepair UK Google Ads account. This list
is **not** customer UI. Copy these groups into each campaign as exact, phrase
and broad matches as appropriate. Review monthly against the search-terms
report.

We are an emergency mobile tyre fitting service operating across Scotland —
not a recruiter, not a training provider, not a retail tyre shop, not a tools
retailer. Anything that signals one of those intents should be excluded.

## Review notes

- Re-run the search-terms report every 30 days and append new negatives.
- Never add a negative that would also block legitimate emergency phrases
  ("emergency tyre", "mobile tyre fitter", "puncture repair near me", etc.).
- Keep this file in sync with the live Ads account. The Ads account is the
  source of truth at run time; this file documents intent.
